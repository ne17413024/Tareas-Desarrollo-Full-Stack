# Ejercicio 1: Suma con Validación

Esta es una aplicación web simple que permite a los usuarios sumar dos números con validación de entrada.

## Archivos

- `index.html`: La estructura HTML principal con entradas para dos números, un botón de calcular y una pantalla de resultados.
- `style.css`: Estilos CSS para un diseño limpio y centrado.
- `script.js`: Código JavaScript que maneja el cálculo, incluyendo validación para campos vacíos y entradas no numéricas.

## Cómo Usar

1. Abre `index.html` en un navegador web.
2. Ingresa dos números en los campos de entrada.
3. Haz clic en el botón "Calcular".
4. El resultado se mostrará abajo, o un mensaje de error si las entradas son inválidas.

## Características

- Validación de entrada: Verifica campos vacíos y asegura que las entradas sean números.
- Interfaz amigable con efectos de hover en el botón.
- Muestra resultados en verde para éxito, rojo para errores.
