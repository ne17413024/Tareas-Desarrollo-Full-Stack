# Ejercicio 2: Calculadora de IMC

Esta es una aplicación web que calcula el Índice de Masa Corporal (IMC) basado en el peso y la altura.

## Archivos

- `index.html`: Estructura HTML con campos para peso y altura, botón de calcular y resultado.
- `style.css`: Estilos para una interfaz centrada y atractiva.
- `script.js`: JavaScript que valida entradas y calcula el IMC con categorías.

## Cómo Usar

1. Abre `index.html` en un navegador.
2. Ingresa el peso en kg y la altura en metros.
3. Haz clic en "Calcular IMC".
4. Verás el IMC calculado y la categoría correspondiente.

## Características

- Validación: Asegura que los valores sean positivos y no vacíos.
- Categorías: Bajo peso (<18.5), Normal (18.5-24.9), Sobrepeso (25-29.9), Obesidad (>=30).
- Interfaz simple con colores para errores y resultados.
